# 🔐 セキュリティ日報 / Security Daily Digest

毎朝7時にClaude（AI）が自動生成するセキュリティエンジニア向け日本語ダイジェストです。

## 概要

- **更新頻度:** 毎日 7:00 JST（自動）
- **対象読者:** セキュリティエンジニア・SOCアナリスト・ペネトレーションテスター
- **情報源:** The Hacker News / BleepingComputer / CISA KEV / JVN（Japan Vulnerability Notes）
- **生成モデル:** Claude Haiku（Anthropic）

## 内容

各ダイジェストには以下が含まれます：

- 🚨 **本日の注目ニュース**（7〜8件）: 積極的に悪用中のCVE、重大インシデント、新しい攻撃手法を優先度順に掲載。各記事に「何が起きたか・なぜ重要か・どう対応するか」の3文要約と元記事リンクを添付。
- 🇯🇵 **JVN国内脆弱性情報**: 国内製品・ソフトウェアの脆弱性をテーブル形式でまとめ。
- 📌 **今日のまとめ**: 全体トレンドの分析と実務への示唆。

## ダイジェスト一覧

`digests/` フォルダに `YYYY-MM-DD_security_digest.md` の形式で日々蓄積されます。
